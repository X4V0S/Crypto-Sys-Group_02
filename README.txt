CryptoParler es una aplicacion que encripta y desencripta mensajes, con la opcion de firmar mensajes.

Fue pensada para tener el papel de "intermediario", es decir, la idea de esta aplicacion es simplemente
hacer el trabajo de encriptar y desencriptar; el medio de difusion de los mensajes encriptados queda a
disposicion de dichos usuarios, mismos que deben estar registrados en la aplicacion para poder hacer uso
de dichas funciones. CryptoParler no es una aplicacion de mensajeria en tiempo real, sino mas bien, es
una aplicacion intermediaria como mencionamos en un principio.

__________________________________________________________________________________________________________

Las dependencias del proyecto se encuentran en el archivo "package.json".
Para correr la aplicacion simplemente se usa el comando "npm run dev" dentro del directorio principal (TRY01).
